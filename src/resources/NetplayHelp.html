Netplay: online multiplayer

In any netplay session, there will be two players present: one will be the server and
the other will be the client. The server will be assigned controller 1 and will pick
the played ROM; the client will be assigned controller 2.

For the server

Select "Netplay -> Local server -> Start" and wait for the client to connect.

For the client

Select "Netplay -> Connect to server" and input the server's hostname and port.
